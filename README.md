bgttoolbox
==========

BrocGlasTech Toolbox scripts and notes.
This Repo doesn't contain anything yet!

## Install

1. Clone the repo
   mkdir -p ~/src; cd ~/src
   git clone git://github.com/Gaoithe/bgttoolbox

2. Put entry in $PATH in ~/.bashrc
   export PATH=$PATH:~/src/bgttoolbox/bin
