Be sure to set the value of UPDATEURL correctly and throw this into your crontab to run every 30 minutes e.g.

Code:

*/30    *       *       *       *       username    /usr/local/bin/dynIP.sh


dynIP.sh:
Code:

#!/bin/sh
CURL=/usr/bin/curl

# get the URL from http://freedns.afraid.org/dynamic/
UPDATEURL='http://freedns.afraid.org/dynamic/update.php?......'

CURRENTIP=`$CURL -s http://ip.dnsexit.com/ | sed -e 's/ //'`
OLDIP=`cat /tmp/ip.cache 2>/dev/null`
if [ "$CURRENTIP" != "$OLDIP" ]; then $CURL -s "$UPDATEURL"; fi
echo $CURRENTIP > /tmp/ip.cache

also, my version (for linux):

Code:

#!/bin/bash

# your internet interface
inet_if=eth0
# curl utility
curl=/usr/bin/curl
# your update url
url=http://..............
# cache file for ip address
ip_cache=/tmp/ipaddr.cache

ifconfig | grep $inet_if || echo "internet interface is down!" && exit 1
current_ip=`ifconfig ${inet_if} | sed '/.*inet addr:/!d;s///;s/ .*//'`
old_ip=`</tmp/$ip_cache`
[ $current_ip != $old_ip ] && \
  $curl -s $url && \
  echo $current_ip >$ip_cache
exit 0

Hi there!
I was looking for a bash script that updates multiple address...
SO i decided to modify the "Guest" script and adapt it.
Eg. if you have 2 links to update, modify the script in this way:
Code:
url_count=2
url[0]=''http://freedns.afraid.org/dynamic/update.php?..."
url[1]="'http://freedns.afraid.org/dynamic/update.php?..."

don't forget to set url_count to the count of links to update!


Code:

#!/bin/bash
inet_if='ppp0'
url_count=1;
url[0]='http://freedns.afraid.org/dynamic/update.php?...'
ip_cache='/tmp/.iptmp'

( ifconfig $inet_if &>/dev/null && test -w $ip_cache ) || exit 0

ADDR=`ifconfig $inet_if | grep inet | sed 's/addr://g' | awk '{print($2)}'`;
OLD_ADDR=`head -n 1 $ip_cache`

echo "Updating IP ($ADDR) at FreeDNS ($OLD_ADDR)..."
if [ "$ADDR" != "$OLD_ADDR" ]; then
        echo "$ADDR" > $ip_cache
   i=0;
   while [ $i -lt $url_count ];
   do
      echo "Updating Address $i of $url_count"
      wget "$url[$i]" -O /dev/null &>/dev/null
      i=$((i+1))
   done
        echo "done"
fi

hello, i'm sorry for bringing this up, but i'd like to show my script too. i think it only runs on freebsd because of the different ifconfig output, but i'm not sure. the ip checking thing in it, that idea i got from this thread. oh and it also supports multiple updates, uses lynx to update and has a verbosity option (-v).

Code:

urls=(url1 url2) #an array
iface="tun0"
lynx="/usr/local/bin/lynx"
sed="/usr/bin/sed"
head="/usr/bin/head"
current_ip=`/sbin/ifconfig $iface | grep inet | /usr/bin/sed -e 's/\-.*//;s/^.//;s/.$//;s/inet //'`
ipfile="/tmp/.ipfile"
old_ip=`$head -n 1 $ipfile`
# End of vars

if [ $current_ip != $old_ip ]; then
     echo $current_ip > /tmp/.ipfile

     if [ -z $1 ]; then
          for i in ${urls[@]}; do
               $lynx -dump $i >/dev/null
          done
     elif [ $1 = "-v" ]; then
          for i in ${urls[@]}; do
               $lynx  -dump $i | $sed -e '/^$/d'
          done
     fi
else
     exit 0
fi
exit 0
